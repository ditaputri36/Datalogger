Panduan Penggunaan

dashboard

Navigasi Halaman

Gunakan menu Home, Elitech, E+Elektronik, dan Benetech di sidebar untuk berpindah antar halaman.

tune

Pilih Kategori

Setiap halaman data memiliki dropdown kategori berbeda. Pilih kategori terlebih dahulu agar data Temperature dan Humidity dapat ditampilkan.

schedule

Filter Waktu

Gunakan filter jam untuk memilih satu atau beberapa waktu pengukuran. Tahan tombol Ctrl untuk memilih lebih dari satu jam.

calendar_month

Filter Tanggal

Pilih tanggal pada kalender untuk menampilkan data sesuai hari yang diinginkan. Tanggal dapat dipilih hingga 25 hari.

date_range

Filter Bulan dan Tahun

Atur bulan dan tahun pada kalender agar data yang dianalisis sesuai dengan periode yang dipilih pada menu Elitech, E+Elektronik, dan Benetech.

analytics

Ringkasan Scorecard

Scorecard menampilkan nilai rata-rata (Avg), maksimum (Max), dan minimum (Min) untuk Temperature dan Humidity berdasarkan filter aktif.

monitoring

Analisis Grafik

Grafik menampilkan data Temperature dan Humidity sesuai page serta kategori yang dipilih. Pada page Elitech, grafik juga menampilkan pembanding E+Elektronik.

table_view

Tabel Summary

Pada setiap page, menampilkan tabel ringkasan data Temperature dan Humidity berdasarkan filter aktif.

open_in_new

Open Google Sheet

Klik tombol Open Google Sheet untuk membuka sumber data Temperature atau Humidity.

download

Export Data

Klik tombol Export Data untuk mengunduh hasil data yang sudah difilter ke dalam format Excel.
